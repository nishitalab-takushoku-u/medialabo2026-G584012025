「Webコンテンツ作成」の成果

本テーマについて

html, CSS, JavaScriptを使ってWebサイトを作る。種類によってデザインが異なり、情報量が多いWebサイトや見た目が綺麗にでき上がれるWebサイトを出来上がれるようにする。また、html、CSS, JavaScript外に出てくる知識を学べる。　たとえば、今の人たちはどこから情報を集めてるかを学ぶ。

プロジェクトの課題について

HTMLで最初レファレンスで引用して真似するWebページを真似して作り上げました。文字を大きくするか太文字にする句小さい文字にするか順番の数字にするかできるような練習をしました。その後に三つの情報符だらけの選択肢があってその中の一つを使った、今回の授業で学んだHTMLの主な要素を使って作り上げなさいの課題でした。選んだのは一番簡単の世界の天気でした。もらった情報全部を使って綺麗なウェブサイトを作るようにしました。

プロジェクト課題で作成したページ: project.html

数当てゲーム

ミニゲームで数字を3回以内に当てないといけないゲーム。毎回当ててみたらウェブからヒントもらい、答え当てたらあたら李真下が書かれる。けれど当てれなかったら残念って言ってくるし、3回以上当てようとisたらもうゲームが終わりましたって伝える数当てゲームです。結構作るのも難しかったです。だけれどプログラムを読んでちゃんと理解すると、わかるようになりました。今まで習ったことを詰めてゲームを作り出す。 ちょっとだけ楽しかったです。けれど難しかったです。 数当てゲーム

第1回 WWW と　HTML

取り組んだ課題はあるWebサイトのページをレファレンスとしてhtmlで主に使われる要素を使って引用したWebサイトを作り上げる。　主に使って要素は「htmlテキスト要素」「ハイパーリンク」「画像の埋め込み」「その他の要素紹介」。その後、使った要素で出来上がった後に、バグがあるかどうかを確認して直せる方法。それは、ページのソースを見るかhtmlバリデーションをしてくれるサイトで確認できます。

演習の成果:演習1-4: HTMLのさまざまな要素

第2回 CSS による Webページの装飾

CSSは作ったウェブサイトを思っと綺麗にできることを学びました。たとえばフォントの形を変えるとか、文字の色を変えるとか、文字を二重にする設定とかもあって。リンク押す前押す後の色とかを設定できます。背景の色を変えることもできて、文字を入れる箱も作り上げてさらに色を入れることもできます。

演習の成果:演習2-3: ウェブブラウザの紹介

第3回 JavaScript の基本

jacascriptでWebサイトをもっと管理できるようないことを学びました。JavaScript は Webの複雑な機能を可能にするプログラミング言語。Web ページを開いた直後や，ボタンを押したときに JavaScript プログラムが実行されます。javascriptでWebサイトに入るときにどのようなリアクションを設定するか、ボタン押した後のアニメーションとかリンク押す前のリアクションとかを作り上がれます。だがcssとhtmlと比べたら一個だけ違う点があって、ビジュアルコードにコードを入力したときコードあっているかどうがわからないことです。間違いコードにならないようにまずファイヤーフォックスでのコンソールでコード確認することができます。

演習の成果:演習3-6:

第4回 DOM と JavaScript

　DOMの略はDocument Object Model、HTML分を釣り扱うためのAPI( Aplication Programming Interface ). つまりjavascriptの複雑な機能にするためにDOMを使う。例えばWebサイトの要素（画像やテキスト）読み込む、書き換える、何かを追加するか消すのかをできる。

演習の成果:演習4-10:

第5回 イベント処理

今回は作ったウェブサイトをもっと面白くなるためにボタンを作り上げて、そのボタンを押した後に変化が起きることを学びました。フィアヤーフォックスのDOMを見ました。（他のウェブサイトになったら結果がたまに少しだけ変わることも教えてもらいました）作るためにはフォーム要素を作り上げた、イベント処理（どのように処理されたい）決めて作ります。

演習の成果:演習5-7

感想

この授業はWebサイトの作り方を学びました。HTML,CSS ,JavaScript、DOMとAJAXの順番で学んで、ファイヤーフォックスでのウェブサイトを作りました。index.htmlは私たちが６回の授業で学んだことを毎回アップデートをしています。毎回内容が変わってて、慣れるために演習をやりました。最初のところは簡単で楽しかったですけれども、どんどん難しくなりました。だけど教授やTAは教えてくれて、できるまで優しく教えてくれました。まずHTMLではどのようなないようにするウェブサイトを作り上げます。ヘッダーや文字の太さや新しい段落に入りたいときはHTMLで調節をしました。さらに6回の中で作り上げるウェブサイトのproject課題の始まりで３個のテーマを選ばせた。テレビ番組、グルメと世界中の天気予報です。一番簡単そうなやつを選びました。天気予報を選びました。可愛いウェブサイトを作り上げれるかなと思って選びましたけれど難しそうでした。次はCSS. 初めてCSSを聴きました。CSSはHTMLが真っ黒なるの避けてくれるプログラムです。書いたた文字の色を指定できて、下線を引くこともできてフォントも決めれて文字のサイズも決めることができます。けれどそれをするためにlinkをしないと書いたプログラムが調査しなくなること。JavascriptはWebサイトをもっと楽しめるWebサイトに作れる。例えばリンク押すときや押した後の色を決めれる。だけどvisual studio書くときコードが間違っている動画を教える赤線がない、そのためコンソールが存在するファイヤーフォックスでコード書いて確認しないといけないです。結構難しく感じました。良いところはjavaと違って string, int, double　を作りたいときは ‘let’だけで宣言できます。短いで便利です。宣言することが簡単だけでなく、プリントするのが簡単でコードを入力できます（console.log）。最後はDOM(1) とDOM(2)、一緒に入れた理由な同じDOMを使っているから一緒にしました。DOMはイベントの処理をすることです。DOMの略はDocument Object Model。オブジェクトを選んで、そのオブジェクトをどうゆうふうに活動したいのかを設定する処理です。ボタンを押したら情報が出してくれるのもできて、For文使って一気に情報を出すこともできます。Functionを作って、何をしたい、足したい、消したいことをしてからaddEventListenerで何をクリックしてからそのFunctionの中を動作するの学びまmした。 まとめての感想ですが、この授業ウェブサイトを開きながら先生の話を聞くところが良かったと思います。PC室で中間モニターもあってとても見やすいで良かったです。先生とTAもみんなとても優しく教えるのも上手です。難しいところもあったけれどとっても楽しかったです。
